Необходимые к установке пакеты: 
* Open CV (`pip install opencv-python`)
* Numpy  (`pip install numpy`)
* PyQt6  (`pip install PyQt6`)
* scipy  (`pip install scipy`)
* screeninfo  (`pip install screeninfo`)
* matplotlib  (`pip install matplotlib`) *использовалось для дебага, возможно уберу, поэтому не надо будет устанавливать*
